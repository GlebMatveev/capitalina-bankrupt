# capitalina-bankrupt

> **commercial development**


Search of debtors by taxpayer identification number.

### Features
- XSLX parsing using PHP and SimpleXLSX
- GET requests using CURL
- output to XLSX
- Integrating in admin panel using Vue 2 with Vuetify
